# 案件管理システム 本番開発 工数見積書

- デモサイト: <https://chiban.vercel.app>

## 1. システム構成

本システムは次の**2つのシステム**で構成します。

1. **案件閲覧システム（営業用）** — 案件・訪問記録の閲覧と、訪問記録の入力を行う
2. **案件入力システム（事務用）** — 案件情報の登録・編集を行う。両システムのユーザー管理を兼ねる

## 2. 見積の前提

- 工数の単位は**人日**です。人日単価は **5万円（税抜）** とします。
- 各機能の工数には詳細設計・実装・単体テストを含みます。要件定義・結合テスト・基盤構築は「共通」に別掲します。
- スケジュール列の日付はすべて **2026年**・1名体制・営業日（土日）ベースの予定です（詳細は「5. スケジュール」参照）。
  **完了済みの項目は「完了」と記載**しています。
